# Netflix Movies and TV Shows Clustering

## Problem Statement
Cluster Netflix movies and TV shows based on metadata and textual features to identify similar content groups.

## Business Use Cases
- Content recommendation
- Market trend analysis
- Content gap identification
- User personalization

## Dataset
- 7787 rows
- 12 columns
- Source: Netflix dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn
- NLP
- TF-IDF
- PCA

## Workflow
1. Data Collection
2. Data Cleaning
3. EDA
4. Feature Engineering
5. TF-IDF
6. Clustering
7. Evaluation
8. Visualization

## Algorithms Used
- K-Means
- Hierarchical Clustering
- DBSCAN

## Evaluation Metrics
- Silhouette Score
- Inertia
- Davies-Bouldin Index

## Results
Successfully grouped Netflix content into meaningful clusters that can be used for recommendation systems.
